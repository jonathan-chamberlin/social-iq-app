# User Research Transcripts

## Naming Convention
`{NNN}_{type}_{date}.md` — e.g. `001_inperson_2026-04-03.md`

- **NNN**: Sequential session number (001–010+)
- **type**: `inperson` or `remote`
- **date**: ISO date of session

## Target: 10 Sessions
- Sessions 001–005: In-person (ZBT watch sessions)
- Sessions 006–010: Remote

## Success Metric
User completes a lesson AND voluntarily starts a second one unprompted.

## After All Sessions
Fill in `synthesis.md` with patterns across sessions.
